# 技术书籍推荐

## JavaScript

### 《JavaScript 高级程序设计》
- 经典的 JS 红宝书
- 涵盖 JS 核心概念
- 适合深入学习

### 《你不知道的 JavaScript》系列
- 深入 JS 内部机制
- 作用域、闭包、this 等
- 进阶必读

## CSS

### 《CSS 揭秘》
- 47 个 CSS 技巧
- 实战案例分析
- 提升CSS技能

### 《CSS 世界》
- 从 CSS 设计理念出发
- 深入理解 CSS
- 国内经典之作

## 框架

### 《Vue.js 设计与实现》
- Vue 3 核心原理
- 霍春阳著作
- 深入理解框架设计

## 设计模式

### 《设计模式：可复用面向对象软件的基础》
- GoF 经典之作
- 23 种设计模式
- 架构师必读
