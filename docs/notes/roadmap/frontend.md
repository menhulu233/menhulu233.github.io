# 前端学习路线

## 第一阶段：基础夯实

### HTML/CSS
- HTML5 语义化标签
- CSS 盒模型、定位、浮动
- Flexbox、Grid 布局
- 响应式设计

### JavaScript
- ES6+ 新特性
- DOM 操作
- 事件处理
- 异步编程

## 第二阶段：框架深入

### Vue.js
- Vue 3 组合式 API
- Vue Router
- Pinia 状态管理
- 组件化开发

### React
- Hooks
- React Router
- Redux/Zustand
- 组件设计模式

## 第三阶段：工程化

- Git 版本控制
- Webpack/Vite 构建工具
- ESLint/Prettier 代码规范
- CI/CD 自动化部署

## 第四阶段：进阶提升

- 性能优化
- 前端安全
- 微前端
- 跨端开发
