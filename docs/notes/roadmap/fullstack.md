# 全栈学习路线

## 前端基础

- HTML/CSS/JavaScript
- Vue.js 或 React
- TypeScript

## 后端开发

### Node.js 方向
- Express/Koa 框架
- NestJS 企业级框架
- RESTful API 设计
- GraphQL

### 数据库
- MySQL/PostgreSQL
- MongoDB
- Redis 缓存

## 运维与部署

- Docker 容器化
- Linux 基础
- Nginx 反向代理
- 云服务（阿里云/腾讯云）

## 项目实战

- 个人博客系统
- 电商后台管理
- 实时聊天应用
